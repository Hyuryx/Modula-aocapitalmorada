# Modula-aocapitalmorada

# 14. Como usar diariamente

Inicie o painel:

npm run painel-avisos

Abra:

http://localhost:3001

Depois:

Preencha o aviso.
Clique em Salvar aviso.
Crie outros avisos, caso necessário.
Clique em Salvar avisos.json.
O arquivo será atualizado.
Um backup será criado.
Envie a alteração ao GitHub.
